Menampilkan Data dalam JavaScript

1. Menggunakan document.write()
Metode ini menulis data langsung ke dalam dokumen HTML (tidak disarankan untuk penggunaan modern).

2. Menggunakan textContent
MtextContent adalah properti yang mengubah konten teks dari elemen HTML tanpa mempengaruhi struktur HTML-nya. Ini lebih aman dibandingkan innerHTML karena tidak memungkinkan eksekusi HTML yang dapat berpotensi membahayakan (seperti XSS).

3. Menggunakan innerHTML.()
Metode ini mengubah konten dari elemen HTML tertentu.
